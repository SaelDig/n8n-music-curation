# N8N Skills

Ce dossier contient des skills Claude personnalisés pour faciliter le développement de workflows N8n de haute qualité.

## Liste des Skills

### 1. n8n-mcp-tools-expert
Expert dans l'utilisation des outils MCP N8n (search_workflows, get_workflow_details, execute_workflow).

**Usage** : `/n8n-mcp-tools-expert`

### 2. n8n-workflow-patterns
Maîtrise des patterns de conception de workflows N8n (trigger-process-store, parallel processing, error handling, etc.).

**Usage** : `/n8n-workflow-patterns`

### 3. n8n-node-configuration
Expert en configuration des nodes N8n (HTTP Request, Webhook, Code, PostgreSQL, etc.).

**Usage** : `/n8n-node-configuration`

### 4. n8n-code-javascript
Spécialiste du code JavaScript dans les nodes N8n (Code node, Function node, expressions).

**Usage** : `/n8n-code-javascript`

### 5. n8n-code-python
Spécialiste du code Python dans les nodes N8n.

**Usage** : `/n8n-code-python`

### 6. n8n-expression-syntax
Expert de la syntaxe des expressions N8n ($json, $node, $env, etc.).

**Usage** : `/n8n-expression-syntax`

### 7. n8n-validation-expert
Expert en validation et debugging de workflows N8n.

**Usage** : `/n8n-validation-expert`

## Comment utiliser ces skills

1. Dans une conversation avec Claude, tape `/skill-name` pour activer un skill
2. Le skill guidera Claude pour t'aider spécifiquement sur ce domaine
3. Tu peux combiner plusieurs skills si nécessaire

## Quand utiliser chaque skill

- **Créer un nouveau workflow** → n8n-workflow-patterns
- **Configurer un node spécifique** → n8n-node-configuration
- **Écrire du code dans un workflow** → n8n-code-javascript ou n8n-code-python
- **Utiliser les outils MCP** → n8n-mcp-tools-expert
- **Résoudre des problèmes d'expressions** → n8n-expression-syntax
- **Debugger un workflow** → n8n-validation-expert
